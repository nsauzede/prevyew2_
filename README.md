# 🦀 prevyew2_

[This is a demo](https://nsauzede.github.io/prevyew2_/) of a full-Rust PWA ([Progressive Web Application](https://en.wikipedia.org/wiki/Progressive_web_app)).

It supports offline-mode and showcases some local sensors.

## Features
- [x] application log viewer (auto-scroll tail-like mini-window)
- [x] simple up-down counter (persistent local-storage)
- [x] accelerometer support (x/y/z acceleration from local sensor)
- [x] orientation support (α/β/ɣ compass/tilt from local sensor)
- [x] geolocation support (GPS coordinates from local sensor)
